# 📝 To-Do App

A clean and responsive To-Do application built using HTML, CSS, and Vanilla JavaScript.

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Responsive design
- Clean user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
todo-app/
│
├── index.html
├── style.css
├── script.js
└── README.md
```
